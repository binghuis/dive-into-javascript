## notebook 启动


1. 安装 `deno jupyter`

    `deno jupyter --unstable --install`

    > 关于 Deno 对 Jupyter 的支持 [Deno 1.37: Modern JavaScript in Jupyter Notebooks](https://deno.com/blog/v1.37)

2. 包管理工具 [pdm 安装](https://github.com/pdm-project/pdm) 
3. 依赖安装
    
    `pdm install`

4. jupyter 启动
    
    `pdm learn`
